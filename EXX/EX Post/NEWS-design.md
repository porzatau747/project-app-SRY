# Design System & Guidelines (Advice IT / Weekly Content Planner)

จากการวิเคราะห์ภาพแบนเนอร์โฆษณาและสื่อโซเชียลมีเดียในโฟลเดอร์ `Template` (เช่น สินค้า Smart Plug, เก้าอี้เกมมิ่ง NUBWO, คีย์บอร์ด EGA, เครื่องสำรองไฟ และคอนเทนต์ IT Tips) สามารถถอดรหัสออกมาเป็นแนวทางสำหรับการออกแบบเว็บแอปพลิเคชันหรือสื่อที่เกี่ยวข้องได้ดังนี้:

## 1. โทนสี / ฟอนต์ (Color Tone & Typography)

### โทนสี (Color Palette)
การใช้สีจะแบ่งเป็น 2 ธีมหลักตามหมวดหมู่ของสินค้าและเนื้อหา:
- **Primary Brand Color (สีหลักของแบรนด์):** 
  - 🔵 **Advice Blue:** สีฟ้า/น้ำเงินเอกลักษณ์ของแบรนด์ (ประมาณ `#0055A5` ถึง `#00A3E0`) ใช้เป็นสีหลักพื้นฐาน
- **Light Theme (สำหรับสินค้าทั่วไป, Smart Home, ออฟฟิศ):**
  - เน้นความสะอาดตา สว่าง มินิมอล
  - พื้นหลังสีขาว `#FFFFFF` หรือเทาอ่อน `#F5F7FA`
  - ใช้สีฟ้าแบรนด์เป็นจุดเด่น
- **Dark/Gaming Theme (สำหรับสินค้าเกมมิ่ง, อุปกรณ์ฮาร์ดแวร์, IT Tips):**
  - เน้นความดุดัน ล้ำสมัย ไฮเทค
  - พื้นหลังสีดำ `#111111` หรือสีกรมท่าเข้ม `#0B132B`
- **Accent/Highlight Colors (สีไฮไลท์เพื่อดึงดูดสายตา):**
  - 🔴 **Neon Red:** (`#FF0000` หรือ `#E60000`) ใช้สำหรับป้ายราคา, แจ้งเตือน, หรือสร้างจุดสนใจสูงสุด
  - 🟢 **Neon Green:** (`#39FF14`) ใช้สำหรับสินค้าบางประเภท (เช่น คีย์บอร์ด)
  - 🩵 **Cyan/Neon Blue:** ใช้ทำเอฟเฟกต์เรืองแสง (Glow) ตัดกับพื้นหลังสีดำ

### ฟอนต์ (Typography)
- **ฟอนต์หลัก (Primary Font):** แนะนำฟอนต์ตระกูล Sans-serif แบบสมัยใหม่ที่อ่านง่ายและดูเป็นทางการแต่วัยรุ่นเข้าถึงได้ เช่น **Kanit**, **Prompt**, หรือ **Noto Sans Thai**
- **Headings (หัวข้อหลัก):** ใช้ตัวหนาพิเศษ (Bold / ExtraBold / Black) ขนาดใหญ่ เพื่อสร้าง Impact ให้กับข้อความ (เช่น "ใครเหมาะกับ...", "เครื่องช้าเพราะ...")
- **Body (เนื้อหาทั่วไป):** ใช้ตัวธรรมดา (Regular / Medium) ขนาดพอดีสายตา
- **Text Styling:** มีการใช้ลูกเล่นกับข้อความสูงมาก เช่น 
  - การทำเงา (Drop Shadow)
  - การเรืองแสงรอบขอบข้อความ (Outer Glow)
  - การตีเส้นขอบข้อความ (Stroke / Outline) เพื่อให้ตัวหนังสือลอยเด่นออกจากภาพพื้นหลังที่ซับซ้อน

## 2. Design System (ระบบการออกแบบ)

- **Vibe & Concept:** "Tech Retail & IT Expert" ผสมผสานระหว่างความน่าเชื่อถือของร้านไอทีชั้นนำ กับความตื่นเต้นเร้าใจของสินค้าเกมมิ่งและเทคโนโลยี
- **Visual Style:** High Contrast (ความเปรียบต่างสูง) ออกแบบมาเพื่อ "หยุดนิ้ว" (Stop the scroll) บนโซเชียลมีเดีย ภาพกราฟิกต้องดูมีมิติ มีแสงเงาชัดเจน
- **Shapes (รูปทรง):** 
  - ใช้ **มุมโค้งมน (Rounded Corners)** อย่างสม่ำเสมอ (Border-radius: 8px ถึง 24px) สำหรับการ์ด, ปุ่ม, และกรอบข้อความ เพื่อความละมุนและทันสมัย
  - สินค้ากลุ่มเกมมิ่งอาจมีการใช้รูปทรงตัดเฉียง (Angled shapes) หรือกรอบแบบไซเบอร์พังก์มาผสมผสานเล็กน้อย

---

## 3. Component (ส่วนประกอบหลัก)

- **Product / Feature Cards:** กล่องแสดงฟีเจอร์หรือการเปรียบเทียบ มักมีพื้นหลังสีขาว (ธีมสว่าง) หรือพื้นหลังโปร่งแสง/ไล่สี (ธีมมืด) พร้อมเงา (Drop shadow) เพื่อให้ดูลอยขึ้นมา
- **Price Badges (ป้ายราคา):** เป็นคอมโพเนนต์ที่โดดเด่นที่สุด มักอยู่ในกรอบสีแดงหรือเขียว ตัวเลขราคาโปรโมชั่นมีขนาดใหญ่มาก และมีราคาปกติที่ขีดฆ่า (Strikethrough) อยู่ด้านล่างหรือด้านข้าง
- **Call to Action (CTA) Buttons:** ปุ่มกระตุ้นการตัดสินใจ เช่น "พิกัด : ADVICE สามร้อยยอด" มักเป็นรูปทรงแคปซูล (Pill-shape) วางเด่นอยู่ด้านล่างสุด
- **Icons & Badges:** 
  - ไอคอนแสดงคุณสมบัติ (เช่น Wi-Fi, แบตเตอรี่, ประกัน) มักวางอยู่ในวงกลมสีพื้นฐานของธีม
  - ตราสัญลักษณ์รับประกัน (Warranty) หรือ "ของแท้ 100%"
- **Callouts / Speech Bubbles:** กล่องข้อความแบบคำพูด ใช้สำหรับเน้น Tips, ชี้แจงปัญหา (Pain points) หรือคำโปรยที่น่าสนใจ

## 4. Layout / Spacing (การจัดวางและช่องไฟ)

- **Visual Hierarchy (ลำดับความสำคัญในการมอง):**
  1. **Headline (หัวข้อหลัก):** ขนาดใหญ่สุด วางตำแหน่งบนสุด หรือจุดที่สายตาโฟกัสแรก
  2. **Hero Image (รูปสินค้า):** ขนาดใหญ่ ตรงกลางภาพ หรือด้านข้างที่โดดเด่น ไดคัทเนียนตา
  3. **Features/Selling Points:** ข้อความสนับสนุน ไอคอน หรือตารางเปรียบเทียบ วางล้อมรอบรูปสินค้า หรือจัดเรียงเป็น Grid
  4. **Price:** วางตำแหน่งที่เห็นชัดเจน มักเป็นมุมขวาล่างหรือใกล้กับตัวสินค้า
  5. **Footer:** แถบด้านล่างสุดสำหรับระบุพิกัดร้านค้าหรือช่องทางการติดต่อ
- **Spacing:** มีการเว้นช่องไฟ (Padding/Margin) ภายในกล่องข้อความอย่างเหมาะสม เพื่อไม่ให้ดูอึดอัด และจัดกลุ่มข้อมูลที่เกี่ยวข้องกัน (เช่น ไอคอนคู่กับข้อความสั้นๆ) ไว้อย่างเป็นระเบียบ
- **Grid Alignment:** มีการใช้ Grid อย่างชัดเจน เช่น การแบ่งฝั่งซ้าย-ขวา (Before-After) หรือการแบ่งคอลัมน์ 3-4 คอลัมน์สำหรับกลุ่มเป้าหมาย/คุณสมบัติ

## 5. Responsive (การรองรับขนาดหน้าจอสำหรับ Web)

เนื่องจากต้นฉบับเป็นภาพสัดส่วน 1:1 หรือ 4:5 สำหรับ Social Media ซึ่งมีลักษณะคล้ายหน้าจอ Mobile อยู่แล้ว เมื่อนำมาประยุกต์ใช้กับ Web Design สามารถกำหนดแนวทางได้ดังนี้:

- **Mobile-First Approach:** บนมือถือ (หน้าจอแคบ) ให้แสดงผลแบบ 1 คอลัมน์ (Stacking) เรียงลำดับจากรูปภาพ Hero -> หัวข้อ -> ฟีเจอร์ย่อย (อาจปัดซ้ายขวาได้แบบ Carousel) -> ราคา -> ปุ่มสั่งซื้อ
- **Desktop Adaptation:** 
  - ขยายเลย์เอาต์ให้เป็น 2 คอลัมน์ (รูปภาพอยู่ซ้าย ข้อความ/ราคาอยู่ขวา)
  - ส่วนที่เป็นรายการฟีเจอร์ (Feature list) หรือกลุ่มเป้าหมาย (Target audience) สามารถกางออกเป็น 3-4 คอลัมน์แนวนอนได้
- **Fluid Typography:** ขนาดฟอนต์ต้องสามารถปรับขนาด (Scale) อัตโนมัติได้ เพื่อให้หัวข้อใหญ่บน Desktop ยังคงดูมีพลัง แต่อ่านง่าย ไม่ล้นจอเมื่ออยู่บนหน้าจอมือถือ
