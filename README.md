# Weekly Content Planner

แอปพลิเคชันสำหรับวางแผนและจัดการคอนเทนต์รายสัปดาห์ของร้านไอที (Advice สามร้อยยอด) แบบครบวงจร โดยผสานการดึงข้อมูลจากระบบหลังบ้านอัตโนมัติ เทรนด์ไอทีล่าสุด พร้อมตัวช่วย AI ในการสร้างตารางโพสต์และ Prompt สำหรับเจเนอเรตภาพโฆษณา

## ความสามารถปัจจุบันของโปรแกรม

### 📦 1. แผนจากสต็อก (Stock Planner)
- **ซิงค์ข้อมูลอัตโนมัติ:** บอท (Playwright) ดึงข้อมูลสต็อกและราคากลางจากระบบร้าน (Nescen) แบบอัตโนมัติ กวาดหาข้อมูลจากทุกหมวดหมู่โดยตรง
- **ระบบตัวกรองหมวดหมู่:** จัดกลุ่มหมวดหมู่สินค้ากว่า 90+ รายการ ให้เหลือ 9 กลุ่มหลัก (Macro Categories) ใช้งานง่ายและดูสบายตา
- **สูตรบวกกำไรอัจฉริยะ (Tiered Markup):** ระบบคำนวณและบวกราคาขายอัตโนมัติตามเรทขั้นบันได (30%, 15%, 10%) และมีข้อยกเว้นพิเศษสำหรับ Notebook (+2,000 บาท) / Monitor, Printer (+300 บาท) พร้อมปัดเศษหลักหน่วยเป็น 0 เสมอ
- **ค้นหาราคาจากหน้าเว็บ Advice:** หากสินค้ารายการใดไม่มีราคากลาง สามารถกดให้บอทไปดึงราคาจริงจากเว็บไซต์ Advice.co.th ได้ และระบบจะบันทึกราคานั้นไว้ถาวร (Persistent Price) 
- **วิเคราะห์สต็อก:** คำนวณ Aging (อายุสินค้าในคลัง) เพื่อวิเคราะห์และคัดหาสินค้าที่ควรเร่งทำโปรโมชันระบายออก (Clearance)

### 📈 2. แผนจากเทรนด์ (Trend Planner)
- **ดึงข่าวอัตโนมัติ (Auto Trend Radar):** กวาดโพสต์ยอดฮิตจากเพจ Facebook ข่าวไอทีชั้นนำ 3 เพจคู่แข่ง (Comcraft, NotebookSpec, Overclockzone) 
- **คัดกรองเฉพาะข่าวที่เป็นกระแส:** ระบบจำกัดผลลัพธ์ดึงเฉพาะข่าวท็อป 4 ของแต่ละเพจ เคลียร์ของเก่าทิ้งทุกครั้งที่เปิดใหม่ เพื่อให้ได้เทรนด์ที่สดใหม่ที่สุด
- **ร่างโพสต์ข่าวด้วย AI ทันที:** เมื่อคลิกเลือกข่าว ระบบจะแต่งแคปชันสรุปข่าวสั้นกระชับ (อ่านจบปุ๊บรู้เรื่องปั๊บ) พร้อม Hook เด็ดๆ
- **สไตล์ภาพข่าวไอทีมืออาชีพ:** สร้าง Image Prompt สำหรับ ChatGPT Images (DALL-E 3) ให้ทันที โดยเรียนรู้การออกแบบจากเพจชั้นนำ (ใช้ Layout แบบ 60/40 Split หรือ Diagonal Overlay พร้อมการเล่นสีข้อความ)

### 🤖 3. สร้างคอนเทนต์ด้วย AI (AI Content Creator)
- **Template หลากหลาย:** รองรับรูปแบบ ทิปส์ไอที (5P/1P), คำแนะนำก่อนซื้อ, โพสต์ขาย/โปรโมชัน, และพิมพ์คำสั่งอิสระ (Free-text)
- **ผสานคู่มือการตลาดท้องถิ่น:** ในโหมดโพสต์ขาย/โปรโมชัน AI จะอ้างอิงคู่มือ `AgentPromotion.md` เพื่อใช้ภาษาแบบเป็นกันเอง เจาะกลุ่มลูกค้าสามร้อยยอด/ไร่ใหม่ สอดแทรกมีมไวรัล และขยี้เทรนด์ไอทีปัจจุบัน
- **Prompt Architecture:** อิงสไตล์การออกแบบ โทนสี และอัตราส่วนภาพ จากเอกสารแนวทางการออกแบบของร้าน (`Stock-Design.md`, `AI-Content-Design.md`, `NEWS-design.md`) ทำให้ได้ภาพที่สวยงามระดับ High-end

### 🎁 4. Promotion Combo (จัดเซ็ตโปรโมชัน)
- **ออกแบบโปรโมชันแลกซื้อ:** คัดเลือกสินค้า 2 ชิ้น (สินค้าหลัก และสินค้ารอง) จากสต็อกมาจัดแคมเปญ
- **AI Content Sales:** แต่งข้อความโปรโมชันกระแทกใจลูกค้าสายคุ้มค่า (เช่น "ซื้อ A แลกซื้อ B ในราคาโคตรถูก") 
- **Localization:** ใช้สำนวนภาษาสำหรับคนพื้นที่ ระบุพิกัดร้านที่ชัดเจน (สาขาไร่ใหม่) พร้อมเชื่อมโยงกับปัญหากวนใจหรือกระแสไอทีในปัจจุบัน

### ✨ 5. ประสบการณ์ผู้ใช้งาน (UX/UI)
- **ดีไซน์ทันสมัย:** เน้น Dark Mode สไตล์เรียบหรู พร้อม UI ที่จัดเรียงอย่างเป็นระเบียบ
- **Toast Notifications:** ระบบแจ้งเตือนสถานะทำงานที่ลื่นไหลผ่าน Pop-up บริเวณมุมจอ
- **Background Automation:** ทำงานรวดเร็วด้วยการเชื่อมต่อ API และแสดงเวลาอัปเดตข้อมูลล่าสุดอย่างชัดเจน

## Tech Stack
- **Frontend / Full-stack:** Next.js, React 18, Tailwind CSS, React Query
- **Backend / AI:** Next.js API Routes, Google Gemini API
- **Data Scraping & Automation:** Playwright (Node.js/TSX), Apify, Cheerio, RSS Parser
- **Utilities:** xlsx (อ่าน/เขียน Excel), react-hot-toast (แจ้งเตือน)
