# Stock Plan Prompt Designs

ไฟล์นี้เก็บรวบรวม System Prompt สำหรับการสร้างคำสั่ง (Prompt) นำไปป้อนให้ AI (ChatGPT / DALL-E) เพื่อวาดภาพโปรโมทโฆษณาสินค้าแต่ละประเภท โดยอ้างอิงจากคลัง Prompt ต้นฉบับที่ใช้งานจริง

---

## 1. หมวดหมู่: Notebook (โน้ตบุ๊ก)

### 1.1 โน้ตบุ๊กทั่วไป (พร้อมของแถม)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพโฆษณาโปรโมทขายโน้ตบุ๊กโดยโชว์จุดเด่นจริงของตัวสินค้าว่าเหมาะกับการใช้งานอะไร 
- สไตล์ตามเทรนด์ไอทีในปัจจุบัน น่าดึงดูด
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุชื่อรุ่น)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - ของแถม: กระเป๋าโน้ตบุ๊ก (ตามแบรนด์) แบบเป้ และ (ระบุของแถมเพิ่มเติม)
  - **หมายเหตุสำหรับ AI:** ให้ค้นหาจากอินเทอร์เน็ต (เว็บ Advice) ว่ารุ่นนี้แถมอะไรมาในเครื่องบ้าง ให้ระบุมาให้ครบ 2 รายการ พร้อมมูลค่าของแถม
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** โชว์ประสิทธิภาพจริงของตัวโน้ตบุ๊ก วิเคราะห์และดึงสเปคจากการค้นหาในอินเทอร์เน็ตมาปรับใช้ในภาพ

### 1.2 โน้ตบุ๊กเกมมิ่ง (พร้อมทดสอบ FPS)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพโฆษณาโปรโมทขายโน้ตบุ๊กเกมมิ่ง สไตล์ตามเทรนด์ไอทีในปัจจุบัน น่าดึงดูด
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุชื่อรุ่น)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** 
  - เน้นโชว์ประสิทธิภาพการเล่นเกม
  - **ฟีเจอร์เด่น:** นำเสนอผลการทดสอบค่าเฉลี่ย FPS ของ 4 เกมฮิตในปัจจุบัน โดยระบุว่าเป็นผลทดสอบแบบ "เปิด DLSS" 
  - ให้ค้นหาสเปคจากอินเทอร์เน็ตและวิเคราะห์ประสิทธิภาพให้สอดคล้องกัน

## 2. หมวดหมู่: Chair (เก้าอี้เกมมิ่ง/สุขภาพ)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพคอนเทนต์โฆษณาโปรโมทขายเก้าอี้เกมมิ่ง สไตล์เทรนด์ไอทีในปัจจุบัน ผสมผสาน "มีมไวรัล (Viral Meme) ในประเทศไทย" ให้ดูไม่รก ดึงดูด กระชับ
- **องค์ประกอบเนื้อหา (Content Breakdown):**
  - **Hook:** วิเคราะห์และตั้งคำถามว่า "ทำไมถึงต้องซื้อสินค้านี้"
  - มี Bridge Content (เชื่อมโยงเข้าหาสินค้า)
  - มี Useful (ให้ประโยชน์)
  - มี Product (นำเสนอสินค้า)
  - มี Meme Angle (ใส่มุกตลก หรือการจัดวางมีม)
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุรุ่นเก้าอี้)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** โชว์ประสิทธิภาพจริงของสินค้า โดยค้นหาสเปคจากอินเทอร์เน็ตแล้วนำมาใส่ในภาพ

## 3. หมวดหมู่: Gaming Gear (คีย์บอร์ด, หูฟัง, เมาส์)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพโฆษณาเกมมิ่งเกียร์ สไตล์เทรนด์ไอที ผสมผสาน "มีมไวรัลในประเทศไทย" แบบกระชับและดึงดูด
- **องค์ประกอบเนื้อหา:**
  - **Hook:** วิเคราะห์จาก "จุดเด่นจากข้อมูลจริงของสินค้านี้"
  - มี Bridge Content
  - มี Product
  - มี Meme Angle
  - มีการแตกคอนเทนต์ (ย่อยเนื้อหาให้อ่านง่าย)
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุรุ่น)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** ค้นหาและวิเคราะห์สเปคจากอินเทอร์เน็ตเพื่อนำมาอธิบายในภาพให้ถูกต้อง

## 4. หมวดหมู่: Speaker (ลำโพง)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพโฆษณาลำโพง สไตล์เทรนด์ไอที มีมไวรัลในประเทศไทย สั้นกระชับ ไม่รก
- **องค์ประกอบเนื้อหา:**
  - **Hook:** วิเคราะห์และตั้งคำถามว่า "ทำไมถึงต้องซื้อสินค้านี้"
  - มี Bridge Content
  - มี Product
  - มี Hook (ดึงดูดอีกครั้ง)
  - มี Meme Angle
  - มีการแตกคอนเทนต์
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุรุ่น)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** ค้นหาและวิเคราะห์สเปคจากอินเทอร์เน็ตเพื่อนำมาใส่ในภาพให้ถูกต้อง

## 5. หมวดหมู่: Printer (ปริ้นเตอร์)
**อัตราส่วนภาพ:** 4:5
**โครงสร้าง Prompt พื้นฐาน:**
- สร้างภาพโฆษณาปริ้นเตอร์ สไตล์เทรนด์ไอที น่าดึงดูด ให้ตัวสินค้าดูโดดเด่น
- **ข้อมูลบังคับ:**
  - ชื่อรุ่น: (ระบุรุ่น)
  - ราคา: ราคาพิเศษ (ลดแล้ว) บาท จากปกติ (ก่อนลด) บาท
  - โปรโมชันเสริม: **"พร้อมหมึกแท้ในกล่อง 1 ชุด"**
  - พิกัด: Advice สามร้อยยอด
- **การออกแบบภาพ:** 
  - เน้นโชว์ฟังก์ชันและประสิทธิภาพความสามารถจริงของเครื่อง
  - ค้นหาสเปคและข้อมูลจริงจากอินเทอร์เน็ตมาปรับใช้ในภาพ
